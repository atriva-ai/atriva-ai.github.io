---
title: Onboarding Checklist
order: 3
---

### **Week 0 – Before Joining**

* [ ] Install Python 3.10+
* [ ] Install Docker
* [ ] Install VS Code + Python extension
* [ ] Join Discord
* [ ] Read `quickstart.md`
* [ ] Run at least **one** test script locally
* [ ] Run the Docker container successfully

### **Week 1 – Getting Started**

* [ ] Pick a **beginner issue** labeled `good first issue`
* [ ] Understand the inference API classes
* [ ] Explore `/tests` folder
* [ ] Join Discord #introductions
* [ ] Ask for help if stuck

### **Week 2 – First Contribution**

* [ ] Fork repo
* [ ] Create feature branch
* [ ] Make small PR (doc fix, test fix, comments, logging)
* [ ] Follow review feedback

### **After Acceptance**

* [ ] Add yourself to CONTRIBUTORS.md
* [ ] Pick next issue
* [ ] Join weekly office hour (if available)
